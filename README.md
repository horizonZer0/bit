# Bit
Bit- the speedy text-to-binary converter.
Bit is available on multiple programmin languages, including JavaScript, PHP, Python, and others.
# Usage
You can use Bit with multple languages. Here are examples for supported ones:
```javascript
Bit("")for text
//or
Bit()//for a value or variable
```
```php
Bit("")for text
//or
Bit()//for a value or variable
```
```python
from bit import Bit

variable = Bit("")
print(variable)
```
